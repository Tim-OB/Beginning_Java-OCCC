PROJECT TITLE: Jump Game
PURPOSE OF PROJECT: Final Project
VERSION or DATE: 0.8
HOW TO START THIS PROJECT: Press the 'Run' button inside Greenfoot
AUTHORS:Tim Weaver and Tim O'Brien
USER INSTRUCTIONS:
	MOVEMENT: To move left and right, press the left and right arrows
		  To jump, press the up arrow
		  To fire gun, press the spacebar

	SCORING : Gain 5 points for picking up a gem
		  Gain 10 points for killing an enemy

	ENDING  : The game ends when the player reaches the treasure chest or falls into the water
		  A game over screen will appear with the players total score when the game ends.
